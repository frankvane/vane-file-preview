第一章：开始

这是一个测试电子书的第一章。这个 EPUB 文件是为了演示 Vane File Preview 组件的 EPUB 预览功能而创建的。

EPUB（Electronic Publication）是一种开放的电子书标准，广泛用于电子书阅读器和各种阅读应用程序。它基于 Web 技术，使用 HTML、CSS 和 JavaScript 来创建可重排的电子书内容。

在这个演示中，您可以看到：

  • 完整的电子书阅读体验
  • 自动生成的目录导航
  • 字体大小调节功能
  • 深色和浅色主题切换
  • 键盘翻页支持
  • 响应式设计

这些功能让用户可以在浏览器中获得接近专业电子书阅读器的体验。
